# Worked Examples

This directory holds worked examples of OrdSA-conformant deployments at varying scales — concrete reference shapes that adopters can read alongside the construct definition.

Each example shows how a real (or canonical) deployment maps to OrdSA's seven ordinal layers, where its authority boundaries sit, how its evidence flow operationalizes O6, and which execution rights its agents are configured at.

## What goes here

Worked examples are descriptive, not normative. They show *one* concrete deployment that satisfies OrdSA's invariants; they do not claim to be the *only* such deployment, and adopters in different contexts will reach different concrete shapes while still being OrdSA-conformant.

Each example is **read against the construct** — it cites OrdSA's layer files and governance principle by reference rather than restating them. The example's job is to make the construct's abstractions concrete, not to redefine the construct.
